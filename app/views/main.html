Instructions

  1. Create a new view, and call it 'shopping-cart.html' in the app/views folder.
  2. Create a controller for this view. Call it 'shoppingCartController' inside the app/scripts/controllers folder. Make sure to add the script to the bottom of app/index.html
  3. Create a route for the view. Call it 'shopping-cart'. The routing config can be found in app/scripts/app.js
  4. Add a link to this route in the header, which can be found in app/index.html
  5. Inject the dataService factory into your controller. This factory can be found in app/scripts/services/dataService.js
  6. Using the cartItems variable from dataService, create a table and use ng-repeat to display the values from the array. Column headings are: Product Name, Product Code, and Price
  7. *BONUS* create a function to total the price values and display it at the bottom of the table.
